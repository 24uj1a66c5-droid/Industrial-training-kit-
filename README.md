# Rule-Based AI Chatbot

## Description
A simple Rule-Based AI Chatbot built using Python. The chatbot responds to predefined user inputs using if-elif conditions.

## Features
- Greets users
- Responds to "How are you?"
- Tells its name
- Exits when user types "bye" or "exit"
- Handles unknown inputs

## Technologies Used
- Python 3

## How to Run
1. Open chatbot.py in VS Code.
2. Run the Python file.
3. Enter messages in the terminal.
4. Type "bye" or "exit" to stop the chatbot.

## Sample Inputs
User: hello
Bot: Hello! How can I help you?

User: how are you
Bot: I am fine. Thank you!

User: name
Bot: My name is AI Chatbot.

User: bye
Bot: Goodbye! Have a nice day.
